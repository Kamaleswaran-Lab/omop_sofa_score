# Tutorial: Running SOFA_on_OMOP

This tutorial walks through a complete SOFA and Sepsis-3 calculation using synthetic OMOP data.

## Files
- 01_load_data.py - loads CDM tables from CSV
- 02_run_sofa.py - computes daily SOFA
- 03_run_sepsis3.py - computes suspected infection and Sepsis-3
- sample_data/ - tiny synthetic OMOP tables for testing

Run in order. Expected runtime < 1 minute on sample data.
